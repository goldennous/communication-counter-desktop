# Communication Counter

## Цель проекта

Веб-приложение для ежедневного учёта количества звонков и чатов в стиле Т-Банка. Пользователь вручную фиксирует показатели за день и видит историю изменений в календаре.

## Основные возможности

### Учёт звонков и чатов

- **Цель:** Быстро фиксировать количество звонков и чатов за текущий день
- **Описание:** На странице «Счётчик» отображаются два счётчика — для звонков и для чатов. У каждого есть кнопки «+» и «-» для увеличения или уменьшения значения. Значения сохраняются автоматически без перезагрузки страницы и без всплывающих уведомлений.
- **Критерии приёмки:**
  - Счётчик звонков и счётчик чатов отображаются на одной странице
  - Кнопка «+» увеличивает значение на 1, кнопка «-» уменьшает на 1 (минимум 0)
  - Значения сохраняются за текущий день
  - Изменение счётчика происходит мгновенно, без перезагрузки страницы
  - Нет всплывающих уведомлений о сохранении
  - Значения сохраняются на правильную дату (текущий день), а не на вчерашнюю

### Просмотр истории по дням

- **Цель:** Видеть, какие значения были записаны за каждый прошедший день
- **Описание:** На странице «История» отображается календарь. Для дней, в которые были внесены данные, показываются значения звонков и чатов.
- **Критерии приёмки:**
  - Календарь отображается на странице «История»
  - Для дней с данными показаны значения звонков и чатов
  - Можно переключаться между месяцами

### Редактирование истории

- **Цель:** Исправить или удалить запись за конкретный день
- **Описание:** При нажатии на день в календаре открывается возможность изменить значения звонков и чатов или удалить запись за этот день.
- **Критерии приёмки:**
  - Клик по дню с данными открывает режим редактирования
  - Можно изменить значения и сохранить
  - Можно удалить запись за день

### Статистика и итоги

- **Цель:** Видеть суммарную статистику за период
- **Описание:** На странице «Счётчик» показываются итоги за текущий месяц. Блок «Декады» разбивает месяц на три периода: 1–10, 11–20, 21–N (N — последний день месяца). В каждом периоде отображается общее количество коммуникаций (звонки + чаты). Текущая декада подсвечена стильным индикатором. Значение окрашивается в цвет: 0–300 — красный, 301–454 — жёлтый, 455 и выше — зелёный. Блок «Месяц» показывает общее количество звонков, чатов и их сумму, а также процент звонков от общего числа коммуникаций за месяц. Процент окрашивается в цвет: 0–29,99% — красный, 30–39,99% — жёлтый, 40% и выше — зелёный.
- **Критерии приёмки:**
  - Блок «Декады» отображает три периода: 1–10, 11–20, 21–N
  - В каждом периоде показано общее количество коммуникаций
  - В блоке «Декады» нет строки «Всего»
  - Текущая декада подсвечена индикатором
  - Значение окрашено в цвет по правилу: 0–300 красный, 301–454 жёлтый, 455+ зелёный
  - Блок «Месяц» показывает звонки, чаты, всего и процент звонков
  - Процент окрашен по правилу: 0–29,99% красный, 30–39,99% жёлтый, 40%+ зелёный
  - Итоги обновляются при изменении данных

## Дополнительные заметки

- Дизайн в стиле Т-Банка: тёмная тема, жёлтый акцентный цвет (#FFDD2D), крупная типографика, плавные анимации, карточки с закруглёнными углами
- Интерфейс адаптивный: приложение корректно отображается в узком окне (от 320px), все элементы остаются читаемыми и функциональными
- Две вкладки (таба) для навигации: «Счётчик» и «История» — расположены по центру шапки
- При открытии приложения сразу отображается страница «Счётчик»
- Все данные хранятся в базе данных (DynamoDB) или локально (localStorage для десктопной версии)
- Интерфейс на русском языке, дружеский тон (ты, нажми, посмотри)
- Тумблер переключения темы: светлая / тёмная
